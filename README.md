# README
## 買取相場検索アプリ

楽天市場販売価格をもとにした相場検索アプリです。
<br>
スマホでの操作を想定しております。


## 作成した目的

相場のわからないお客様に対し、不安を取り除くことを期待し作成いたしました。
<br>
査定スタッフ、お客様双方が共通のデータを見ることで、金額説明の根拠に説得力を持たせます。
